# Сценарий скринкаста (2:30 - 3:00 мин)

## Подготовка

**Окна (расположить заранее):**
- ЛЕВОЕ: браузер Firefox/Chrome на http://localhost:3000/
- ПРАВОЕ: терминал PowerShell с прогретым `uvicorn` (бэкенд) - для случая, если жюри захочет увидеть API-логи

**Перед записью:**
1. Перезапустить бэкенд (чтобы in-memory был чист, `worksheets:0` в /health)
2. В браузере открыть http://localhost:3000/, дождаться загрузки главной
3. Закрыть все панели разработчика и уведомления
4. Подключить микрофон, проверить уровень
5. (Опционально) открыть в фоне два DOCX из `data/exports/` на случай если live-скачивание упадёт

**Запись:** OBS Studio → preset 1080p / 30fps / 4Mbps. Window Capture окна браузера.

**Действие:** клик по чипу «обыкновенные дроби» (если в 6 классе) или вписать тему, выставить класс 6, задач в варианте 2

> «Например, обыкновенные дроби, 6 класс, 2 задачи в варианте.»

### Сцена 3. Второй блок другого предмета (0:55 - 1:25)

**Действие:** клик «Добавить блок»

> «Главная фишка: одну контрольную можно собрать из разных тем и предметов. Добавим второй блок.»

**Действие:** во втором блоке - клик «физика», выбрать класс 7, тема «плотность вещества», 2 задачи

> «Блок 02: физика, 7 класс, плотность вещества, ещё 2 задачи. Итого в каждом из 4 вариантов будет 4 задачи: 2 по математике и 2 по физике.»

**Действие:** клик «Сгенерировать»

> «Под капотом: для каждого блока система ищет в библиотеке программы подходящий эталон через stem-поиск, устойчивый к падежам, GigaChat анализирует эталон и возвращает шаблон со слотами и формулой ответа, sympy подставляет значения и точно считает каждый из 16 вариантов.»

(Пока ждём 30-60 секунд - голос продолжает поверх loading-спиннера, объясняет что происходит)

**Подсветить курсором** sympy ✓ на нескольких карточках, ответы и подставленные значения

## Что заранее показать в окне Word (для подстраховки)

Открыть **до записи** два DOCX-файла из `data/exports/` (например, mixed-students.docx и mixed-teacher.docx из смешанного смоук-теста):
- `mixed-students.docx` - на странице 1 виден заголовок «обыкновенные дроби (6 кл) + плотность вещества (7 кл)», Вариант 1 с двумя подзаголовками блоков, задачи 1-2 матем, 3-4 физика, место для ФИО
- `mixed-teacher.docx` - метаинфо по блокам с эталоном и формулой, потом Вариант 1 с ответами, подставленными значениями и sympy-метками

Так если live-скачивание упадёт - переключаемся Alt+Tab на уже открытый Word.
